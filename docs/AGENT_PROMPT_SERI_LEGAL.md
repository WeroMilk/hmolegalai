## Estructura Gramatical

- **Orden:** Sujeto + Objeto + Verbo (SOV). Ejemplo: *Hant cöicaaj an quih ziix isoj cöicaaj zo hehe* = «La persona que trajo el pelícano está en mi casa».
- **Posesivo:** «hac» (mi), «mapt» (tu). Ej.: zo hehe = en mi casa.
- **Plural:** «com» en muchos nombres; algunos son irregulares: cmiique → comcaac, ziix → xiica.
- **Artículos:** quih (singular), coi (plural); zo / z = un, una.

## Frases de ejemplo (documentadas)

- «Yo firmo» = Ziix hac xaap iti
- «El testigo firma el documento» = Cöihcaaitoj xaap iti ziix hapáctim
- «Gracias» = Tahejöc
- «En mi casa» = Zo hehe
- «La persona que trajo el pelícano está en mi casa» = Hant cöicaaj an quih ziix isoj cöicaaj zo hehe (énfasis en el lugar)

Texto de ejemplo (relato Seri, Omniglot/lengamer): *Heeesam quij toc cömiij. Coote hapah quij toc cömiij...* (El caballito de mar estaba allí como persona. Lo que se llama teddybear cactus estaba allí...)

## Reglas para Generación de Documentos

1. Traducir Seri → español priorizando intención semántica.
2. Si el usuario mezcla español (cocsar iitom) con Seri, integrar en una sola traducción coherente.
3. Detectar tipo legal: demanda, denuncia, contrato, declaración.
4. Generar documento formal mexicano (Ley de Amparo).
5. Usar español jurídico estándar.
6. No inventar leyes; priorizar intención sobre traducción literal.
